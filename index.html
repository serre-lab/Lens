    LENS Project

    Thomas Fel, Thomas Serre
    {thomas_fel}@brown.edu

    Carney Institute for Brain Science, Brown University (Boston, USA),
    DEEL Team - Artificial and Natural Intelligence Toulouse Institute

    This project is the result of several articles, the most notable ones being

    The goal of this project is to use state-of-the-art explainability tools to explore the extent to which we can explain a ResNet50 trained on ImageNet. Each class in ImageNet has its dedicated page where you can see the concepts the model uses to classify that class. For each concept, a normalized importance level is assigned, indicating how crucial the concept is for the class. For instance, an importance level of 0.30 means that a concept contributes to 30% of the sum of logits for all points classified as such. The "Getting Started" page serves as an introduction, explaining how to interpret the results.
